Displays

Sirve para indicarle al navegador cómo debe comportarse cada elemento de la página. A continuacion se mencionan algunos de los valores mas importantes de esta propiedad

1. display: inline - Elementos en Línea

Los elementos con display: inline funcionan de la misma manera que las palabras en una frase. Se alinean en la misma línea, uno tras otro, sin comenzar una nueva línea.

Ejemplo:

Este es un enlace en línea y este es un texto dentro de un span.

2. display: block - Elementos en Bloque

Son como cajas que ocupan todo el ancho disponible y siempre comienzan en una nueva línea. Elementos como div, h1, p, y ul son por defecto de bloque.

Ejemplo:

Este es un elemento de bloque y ocupa toda la línea.

Este es otro elemento de bloque que también comienza en una nueva línea.

3. display: inline-block - Lo Mejor de Dos Mundos

Con display: inline-block, los elementos se alinean uno tras otro como inline, pero también respetan propiedades como width, height, margin, y padding, como un elemento de bloque. Son ideales para menús de navegación o tarjetas en un diseño en fila.

4. display: flex - Flexibilidad en la Distribución

Permite una distribución flexible de los elementos dentro de un contenedor. Se pueden alinear y distribuir elementos fácilmente dentro del contenedor, ideal para diseños más complejos.

Ejemplo:

5. display: grid - La Rejilla Maestra

Es ideal para crear diseños complejos con filas y columnas. Funciona para estructuras de página más avanzadas donde se necesita colocar elementos en una cuadrícula de filas y columnas.

6. display: none - Desapareciendo en Acción

Cuando se utiliza display: none a un elemento, este desaparece completamente de la página, no ocupa espacio y no es visible. Es diferente de visibility: hidden, que solo hace invisible el elemento pero sigue ocupando su espacio.
